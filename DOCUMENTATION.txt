Creating a game object:
Initialize it first with a name, e.g (struct GameObject NAME_HERE)
use the function CreateObject() to set the values
call render object with the GameObject in the args
to "free up memory", call DestroyObject(MEMORY LOC FOR GAMEOBJECT)
